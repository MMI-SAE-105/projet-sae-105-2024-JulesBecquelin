Glossaire

Le courant continu (CC)

est un type de flux électrique dans lequel les électrons circulent dans une seule direction, de manière constante, d'un point à un autre. Contrairement au courant alternatif, le courant continu garde une tension stable dans le temps. Il est utilisé dans des dispositifs comme les piles, les batteries, et certains appareils électroniques.

Le courant alternatif (CA)

est un type de flux électrique où la direction du mouvement des électrons change périodiquement, alternant entre une direction et son opposée. Ce type de courant est utilisé dans les réseaux électriques modernes car il est plus efficace pour transporter l'électricité sur de longues distances. Les prises domestiques fournissent généralement du courant alternatif.
